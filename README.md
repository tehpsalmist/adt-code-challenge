[Check it out live](https://adt-code-challenge.now.sh)

Spin it up locally:

1. Clone and Install

    ```sh
    git clone $url && cd adt-code-challenge
    npm i
    ```

2. Create Environment File

    ```sh
    touch .env
    echo "API_KEY='put api key here'" >> .env
    ```

3. Start Server

    ```sh
    npm start
    ```

    Visit http://localhost:1234 to see it in action!
